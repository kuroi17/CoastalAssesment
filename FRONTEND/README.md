# 🌐 Coastal Assessment Dashboard Frontend

## Overview

A simple HTML/CSS/JavaScript dashboard for visualizing coastal change analysis data from 2020–2025.

## Features

- **Year Selector**: Choose any year from 2020–2025
- **Area Statistics**: View km² measurements for each coastal class
- **Trend Chart**: Multi-year line graph showing changes over time
- **Ecological Summary**: Text analysis of coastal health trends
- **Classification Maps**: Visual representation of coastal coverage

## How to Use

### 1. Generate Data from Notebooks

First, run the notebook to export JSON data:

```bash
# In VS Code or Jupyter, open and run:
notebooks/06_export_frontend_data.ipynb
```

This creates three JSON files in `frontend/data/`:

- `area_stats.json` - Area measurements by year
- `trend_data.json` - Time-series data for charts
- `eco_summaries.json` - Text summaries for each year

### 2. Open the Dashboard

Simply open `index.html` in a web browser:

```bash
# Option 1: Double-click index.html in File Explorer

# Option 2: Use a local server (recommended for fetching JSON)
# Install Python's http.server if needed, then:
cd frontend
python -m http.server 8000
# Open http://localhost:8000 in browser
```

**Note**: Some browsers block `fetch()` requests for local files. Using a local server (Option 2) prevents this issue.

### 3. Interact with Dashboard

- Use the **year dropdown** to select different years
- View area statistics, maps, and trends
- Download PDF reports (placeholder feature)

## Customization

### Add Real Classification Maps

1. Export maps as PNG from notebooks (e.g., `classification_map_2023.png`)
2. Save them as `data/map_2020.png`, `data/map_2021.png`, etc.
3. Dashboard will automatically load year-specific maps

### Modify Color Scheme

Edit `style.css`:

- `.seagrass { background: #1a936f; }` - Seagrass color
- `.sand { background: #ffe156; }` - Sand color
- `.water { background: #0077b6; }` - Water color

### Add More Years

1. Update notebook 06 to include new years
2. Re-run to regenerate JSON files
3. Add year options in `index.html`:
   ```html
   <option value="2026">2026</option>
   ```

## Troubleshooting

### "No data available for this year"

- Run notebook 06 to generate JSON files
- Check that files exist in `frontend/data/`

### Charts not showing

- Ensure Chart.js CDN is accessible
- Check browser console for errors

### Maps showing placeholder

- Export classification maps from notebooks
- Save as `data/map_YYYY.png` format
